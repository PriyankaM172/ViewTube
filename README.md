# ViewTube - YouTube Clone

A responsive YouTube-style video browsing UI built with HTML, CSS, and JavaScript.

## Features

- YouTube-inspired top navigation bar
- Collapsible sidebar menu
- Search videos by title, channel, or category
- Filter videos with category chips
- Responsive video card grid
- Featured video hero section
- Mock video player modal
- Dark mode toggle
- Mobile-friendly layout

## Folder Structure

```text
viewtube/
|-- index.html
|-- css/
|   |-- style.css
|-- js/
|   |-- app.js
|-- assets/
|-- README.md
```

## How To Run

Open `index.html` in any modern browser.

No build tools or package installation are required.

## Tech Used

- HTML5
- CSS3
- JavaScript

## Notes

This is a frontend clone for learning and practice. It does not use the real YouTube API, user accounts, uploads, or actual video playback.

## Possible Improvements

- Add real embedded videos
- Add login and profile pages
- Save dark mode preference in localStorage
- Add like, subscribe, and comment interactions
- Connect the UI to a video API
